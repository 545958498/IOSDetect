# IOSDetect

HOOK检测，可以用来检测插件 HOOk了哪些系统函数, 或者测试越狱商店上那些所谓的越狱屏蔽插件的有效性。

IOS 设备信息检测

1、越狱检测：jab字段,true为越狱 <br>
2、重打包检测: sig字段:false 为重打包 <br>
3、inline hook检测 (详细)<br>
4、fishhook检测   (详细)<br>
5、message hook 检测  (详细)<br>
6、资源patch 检测  (详细)<br>
7、插件检测 <br>
8、按键精灵检测 <br>
9、其余各种检测自己查看字段 <br>

使用说明: 越狱设备安装后，在设置里面选择要注入的APP , 打开APP后双指长按，弹出窗口点击 设备环境信息
